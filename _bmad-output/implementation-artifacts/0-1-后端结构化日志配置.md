# Story 0.1: 后端结构化日志配置

Status: ready-for-dev

## Story

As a 后端开发者,
I want 后端应用输出结构化、分级的日志，并将关键运行事件持久化到文件,
so that 生产环境排查问题时能快速定位错误根因，无需依赖 console.log 散点调试。

---

## Acceptance Criteria

**Given** 后端服务启动  
**When** 任意 HTTP 请求到达  
**Then** 控制台输出包含：时间戳、日志级别、模块名、请求方法、路径、响应状态码、耗时(ms)

**Given** 后端服务运行中  
**When** 发生未处理异常（500 错误）  
**Then** 日志输出 ERROR 级别，包含完整 stack trace

**Given** 生产环境（NODE_ENV=production）  
**When** 服务运行  
**Then** 日志以 JSON 格式输出到 stdout（供 Docker 日志驱动收集）  
**And** 同时写入 `logs/app.log`（按日滚动，保留 14 天）

**Given** 开发环境（NODE_ENV=development）  
**When** 服务运行  
**Then** 日志以可读的彩色文本格式输出到控制台

**Given** 现有 `HttpExceptionFilter`  
**When** 捕获到异常  
**Then** 使用统一 Logger 实例记录，不再使用 `console.error`

**Given** 现有 `ResponseInterceptor`  
**When** 请求完成  
**Then** 记录请求日志（方法 + 路径 + 状态码 + 耗时）

## Dev Notes

### 技术选型依据

架构文档选定"NestJS 内置 Logger（结构化输出到 stdout）"，但内置 Logger 不支持 JSON 格式和文件滚动。`nestjs-pino` 是 NestJS 生态中最成熟的 Pino 集成方案，完全兼容 NestJS Logger 接口，零侵入替换。

### 关键约束

- **不引入 Winston**：架构文档未提及，且 Pino 性能更优（异步写入）
- **保持 `HttpExceptionFilter` 响应格式不变**：只替换日志实现，不改响应结构
- **`LoggingInterceptor` 注册顺序**：必须在 `ResponseInterceptor` 之前，否则拿不到真实状态码
- **Migration 时间戳**：本 Story 无数据库变更，无需创建 migration

### 现有代码参考

- `apps/api/src/common/interceptors/response.interceptor.ts` — 现有拦截器模式
- `apps/api/src/common/filters/http-exception.filter.ts` — 现有 Logger 使用方式（`new Logger(name)`）
- `apps/api/src/app.module.ts` — APP_INTERCEPTOR / APP_FILTER 注册方式

### nestjs-pino 关键用法

```typescript
// app.module.ts
import { LoggerModule } from 'nestjs-pino';

LoggerModule.forRootAsync({
  useFactory: () => loggerConfig(), // 从 logger.config.ts 导入
})

// 注入方式
import { PinoLogger, InjectPinoLogger } from 'nestjs-pino';

@Injectable()
export class LoggingInterceptor {
  constructor(@InjectPinoLogger(LoggingInterceptor.name) private readonly logger: PinoLogger) {}
}
```

### 文件滚动（生产环境）

```typescript
// 使用 pino 内置 transport 多目标
transport: {
  targets: [
    { target: 'pino/file', options: { destination: 'logs/app.log' }, level: 'info' },
    { target: 'pino-pretty', options: { colorize: false }, level: 'warn' }, // 可选
  ]
}
```
